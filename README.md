**摘要**

 - 作者的观点 - Tether不太可能通过任何自然业务发展。它很有可能是根据市场情况进行印钞进行盈利。
 - Tether增发会显著地推动市场发展。在91次不同的Tether到Bitfinex钱包的发放中，**48.8%**的情况比特币的价格会在两小时内上涨。
 - Bitfinex的存取款数据不同寻常，并且会在典型的会计环境中引发进一步的审查。
 - 如果有可疑活动发生，作者认为可以预测比特币的价格会降低**30-80%**。
